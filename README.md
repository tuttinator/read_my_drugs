# Read My Meds

To get setup:

1. Clone this git repo
2. `bundle install`
3. [Register for a Bing translation API](https://datamarket.azure.com/dataset/1899a118-d202-492c-aa16-ba21c33c06cb)
4. Copy .env.sample to .env and change the values
5. ...
6. Profit
